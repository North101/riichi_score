# riichi_score

A Riichi Mahjong scoring and hand analysis library written in Dart.

The library can analyze Mahjong hands, determine valid hand structures, detect yaku and yakuman, calculate han and fu, and compute the final score according to Riichi Mahjong rules.

It supports common rule variations such as red fives, honba, dealer/non-dealer scoring, and both ron and tsumo wins.
